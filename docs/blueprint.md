# **App Name**: ScholarSite

## Core Features:

- Introduction Page: Landing page introducing the student and showcasing key areas of interest and expertise.
- Research & Teaching: Dedicated section for research projects and teaching experience, with links to downloadable papers and project details.
- Projects Showcase: Showcase personal projects with descriptions and visuals.
- Related Courses: Page dedicated to related courses with syllabus information available for each course.
- Contact Form: Contact page with a contact form.
- Skills and Media: Gallery for showcasing media and a brief description of the user's skillset and interest

## Style Guidelines:

- Primary color: Muted teal (#A8DADC) for a calm and professional feel.
- Background color: Off-white (#F1FAEE) for a clean and readable design.
- Accent color: Dark red (#E63946) for highlighting key information and calls to action.
- Body text: 'Inter', a sans-serif font, providing a neutral and modern reading experience.
- Headline text: 'Space Grotesk', a sans-serif font to provide clear headings for easy understanding.
- Clean, simple icons to represent different sections and skills.
- Well-structured, responsive design to ensure readability on all devices.
